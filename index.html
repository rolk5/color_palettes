Дорожно-климатическое районирование

Карта

Информация

Дорожно-климатическое районирование(ДКР) — район (территория) с однородными климатическими условиями, такими как степень увлажнения, глубина залегания грунтовых вод, глубина промерзания грунтов, среднегодовое количество осадков и т.п.

Описание

Для определения дорожно-климатической зоны (ДКЗ) проектируемого объекта используются данные из открытых источников, которые предоставляются в виде статических карт низкого разрешения.
Ниже приведено изображение ДКЗ из ОДМ 218.2.094.

Дорожно-климатическое районирование ОДМ 218.2.094

Определение информации по данному изображению вызывает затруднения. Возникает необходимость в разработке интерактивной карты, которая поможет пользователям лучше ориентироваться и получать более точные данные

Процесс

Источником границ ДКЗ выбрано изображение из ОДМ 218.2.094.
Извлечем границы зон в соответствующих координатах и перепроецируем их в привычную систему координат.
